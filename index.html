關於 JSDC

JSDC (JavaScript Developer Conference) is Taiwan's largest annual technical conference for JavaScript developers. It aims to provide a platform for sharing the latest JavaScript technologies and fostering collaboration among developers. Founded in 2011 by three developer communities, JSDC has grown significantly, bringing together top international speakers and local talent. The conference focuses on cutting-edge topics such as full-stack web development, software and hardware integration, and enterprise JavaScript adoption. JSDC also plays a key role in supporting and promoting various front-end and back-end technology communities in Taiwan.
